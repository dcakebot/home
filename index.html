Commands.

CakeBot only uses slash (/) commands. There is no need to get confused with prefixes!
Fields with [] are required, {} are optionals.

Command Description Usage
/player info Get various information about a Minecraft player. /player info [player name]
/player head Get the head of a minecraft player skin. /player head [player name]
/player skin Get the skin of a minecraft player. /player skin [player name]
/server See the status of a minecraft server. /server [server ip]
/mob Sends a random minecraft mob image. /mob
/animal Sends a random animal picture!. /animal [cat,dog,fox]
/invite Invite CakeBot to your server! /invite
